**Dev Weekends Pomodoro Timer** is a modern, responsive productivity application built with **React** and **Vite** that helps users stay focused using the **Pomodoro Technique**.

The application provides an intuitive timer experience with clean visuals, keyboard accessibility, responsive layouts, and smooth session management designed for students, developers, and productivity enthusiasts.

## ✨ Features

### ⏱️ Pomodoro Timer
- 25-minute focused work sessions
- Dedicated short break mode
- Dedicated long break mode
- Real-time countdown updates
- Start / Pause / Reset controls

---

### 🎨 Visual Experience
- Circular animated timer UI
- Color-coded session states
  - Focus session
  - Short break
  - Long break
- Minimal and distraction-free design
- Smooth visual transitions

---

### 📱 Responsive Design
- Mobile-friendly layout
- Optimized for small screens (360px)
- Enhanced spacing and scaling for large displays (1440px+)
- Flexible component sizing

---

### ♿ Accessibility
- Full keyboard navigation support
- Focusable interactive controls
- Semantic button structure
- Clear visual feedback states

---

### ⚡ Performance Improvements
- Accurate interval handling
- Reduced timer drift
- Efficient React state updates
- Lightweight implementation with no external UI libraries

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/kashmalaasif/Dev-Weekends.git
```

---

## 2️⃣ Move into Project Folder

```bash
cd Dev-Weekends
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start Development Server

```bash
npm run dev
```

---

## 5️⃣ Open in Browser

```text
http://localhost:5173
```
